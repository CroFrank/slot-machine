### slot-machine

My first app using React

Hosted on github, you can visit [here](https://crofrank.github.io/slot-machine/)

# About app

Simple slot machine where you need to get 3 of a same kind in order to win.

# I used:

- Vite for a build tool
- React framework

# Try it out

git clone
npm i
npm run dev
